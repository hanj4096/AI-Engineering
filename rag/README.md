# RAG - 检索增强生成

检索增强生成（Retrieval-Augmented Generation, RAG）技术的学习资料库，涵盖 RAG 架构、向量数据库、检索策略、应用实践等核心主题。

## 📚 项目简介

本目录专注于 RAG 技术的学习与实践，深入解析如何结合检索和生成来提升大语言模型的知识准确性和实时性。内容涵盖从基础架构到高级优化的完整知识体系，适合 AI 工程师、RAG 开发者、知识库系统开发者等技术人员学习和参考。

## 📖 核心内容

### 1. RAG 基础

**主要内容：**
- **RAG 概念** - 什么是 RAG、为什么需要 RAG
- **RAG 架构** - 基本架构、组件设计、数据流
- **RAG vs 微调** - 对比分析、适用场景、选择策略
- **RAG 优势** - 知识更新、减少幻觉、可解释性

### 2. 文档处理

**主要内容：**
- **文档加载** - PDF、Word、Markdown、网页等格式
- **文档分块** - 分块策略、块大小、重叠处理
- **文档预处理** - 清洗、去重、结构化、元数据提取
- **多模态文档** - 图像、表格、代码的处理

### 3. 向量化与嵌入

**主要内容：**
- **嵌入模型** - 文本嵌入、多模态嵌入、领域嵌入
- **向量化策略** - 文档向量化、查询向量化、混合向量化
- **嵌入优化** - 模型选择、微调嵌入、维度优化
- **向量质量** - 向量评估、相似度计算、质量提升

### 4. 向量数据库

**主要内容：**
- **向量数据库选型** - Pinecone、Weaviate、Milvus、Qdrant
- **索引技术** - 倒排索引、LSH、HNSW、IVF
- **检索优化** - 近似最近邻、精确检索、混合检索
- **数据库管理** - 数据更新、版本管理、备份恢复

### 5. 检索策略

**主要内容：**
- **检索方法** - 密集检索、稀疏检索、混合检索
- **重排序** - 相关性重排、多样性重排、交叉编码器
- **多跳检索** - 迭代检索、图检索、推理检索
- **检索优化** - 查询扩展、查询重写、上下文检索

### 6. 生成与融合

**主要内容：**
- **提示构建** - 上下文注入、提示模板、提示优化
- **生成策略** - 单次生成、流式生成、多候选生成
- **结果融合** - 多检索结果融合、多模型融合
- **后处理** - 结果过滤、格式标准化、引用生成

### 7. 高级 RAG 技术

**主要内容：**
- **Self-RAG** - 自我检索、自我评估、自适应检索
- **Corrective RAG** - 错误纠正、迭代改进
- **GraphRAG** - 图结构知识、图检索、图推理
- **Multi-Agent RAG** - 多智能体协作、分工检索

### 8. 应用实践

**主要内容：**
- **知识问答** - 企业知识库、FAQ 系统、智能客服
- **文档分析** - 文档摘要、文档问答、文档对比
- **代码助手** - 代码检索、代码解释、代码生成
- **研究助手** - 论文检索、文献综述、研究分析

### 9. 性能优化

**主要内容：**
- **检索优化** - 检索速度、检索精度、检索召回
- **生成优化** - 生成速度、生成质量、生成成本
- **系统优化** - 缓存策略、批处理、异步处理
- **成本优化** - API 成本、存储成本、计算成本

### 10. 评估与监控

**主要内容：**
- **评估指标** - 准确性、相关性、完整性、时效性
- **评估方法** - 人工评估、自动评估、A/B 测试
- **监控系统** - 性能监控、质量监控、异常检测
- **持续改进** - 反馈收集、模型迭代、系统优化
